# کمی شعر کهن بخوانیم

- شعرهای کهن فارسی را راحت تر بخوانیم و به اشتراک بگذاریم .
- ابزاری ساده برای بازطراحی وبسایت گنجور و بهتر خواندن اشعار کهن فارسی
