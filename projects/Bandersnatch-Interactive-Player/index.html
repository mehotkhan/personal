# Bandersnatch Interactive Player (based on html5 video player)

With this online(html) video player you can watch Bandersnatch episode of Black Mirror interactively.

Live demo (opens new window)

# How to use

The only thing you need is 5hours video file of Bandersnatch episode, Drag & drop the file in this (opens new window) page and enjoy!
